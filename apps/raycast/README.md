# Omi for Raycast

Save and search your [omi](https://app.omi.ac) resources directly from Raycast.

## Commands

- **Search Resources** — Browse your saved websites, notes, and files in a visual grid with og-image previews. Search by title.
- **Save URL** — Save any URL to your omi workspace.
- **Save Current Tab** — Capture the active browser tab in one keystroke (requires the [Raycast Browser Extension](https://www.raycast.com/browser-extension)).
- **Save Note** — Quickly save a plain-text note.
- **Connect Omi** — Link Raycast to your omi account.

## Setup

1. Run **Connect Omi**. Your browser will open a sign-in page.
2. Sign in to omi and approve the connection.
3. You'll be redirected back to Raycast — you're done.

You can revoke access any time from omi's settings.

## Self-hosting

If you run a self-hosted omi deployment, change the **Omi App URL** and **Omi API URL** preferences to point to your deployment.
